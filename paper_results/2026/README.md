Paper Results (2026)

This directory contains all figures and datasets used in the 2026 paper
"A Time-Bounded, Agent-Mediated, Fail-Secure Communication Architecture
for Intelligent Power Systems".

figures/  : Final PNG figures used in the paper
csv/      : Raw CSV data used to generate figures and tables
tables/   : Derived tables (if applicable)

No post-processing, smoothing, or curve fitting is applied.
All figures are generated directly from simulation outputs.
